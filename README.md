### 1 功能分析

1. 登录注册
2. 发帖
3. 审帖
4. 搜索帖子
5. 各种排序，按照时间、按照热门程度等等
6. 点赞、评论、点踩
7. 加标签

### 2 技术栈

- mysql
- springboot
- mybatis
- springmvc
- redis
- rest风格的接口设计
- junit测试

### 3 计划

1. 设计数据库`er`模型的实现 + 表的设计

2. `POJO`的生成 + `Mapper`层 + `Service`层 + `ServiceTest`

3. `Controller`层 + `HttpClient`测试
